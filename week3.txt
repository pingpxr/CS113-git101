ฟังก์ชัน if,else,else if,while,for ควรใส่ ปีกกาเสมอ {}
เพื่อกำหนด block ของมัน และมักจะใช้ block เป็นการรวม statement หลายๆอัน
if ใช้ในการสร้างเงื่อนไขว่าเป็นจริงหรือไม่
ส่วน else if จะเป็นการเพิ่มเงื่อนไขขึ้นมาให้มีหลายทางเลือกทำให้ ถ้าเงื่อนไขแรกไม่ผ่านก็จะไปเงื่อนไขต่อไป จนกว่าจะเจอเงื่อนไขเป็นจริง 

switch
จะทำงานคล้ายกับ if แต่จะใช้ในการตรวจสอบเงื่อนไขว่าตรงกับค่าคงที่ และต้องเป็นข้อมูลประเภท Integer หรือ Char จะทำงานเรื่อยๆจนกว่าจะพบ break 

while loop
จะทำงานไปเรื่อยๆจนกว่าเงื่อนไขจะเป็นเท็จ

for loop
สามารถกำหนดค่าเริ่มต้นได้
for (;;){
    printf("hello\n");
}
    จะเป็นการพิมพ์ "hello" infinite

do while loop
ทำงานคล้าย while จะทำงานหนึ่งรอบ ก่ออนที่จะตรวจว่าเงื่อนไขเป็นจริงหรือไม่ หากเป็นจริงจะวนจนกว่าจะเป็นเท็จ

break และ Continue
break จะหยุดการทำงานของลูปทันที
continue ทำให้ลูปวนซ้ำขึ้นไปใหม่ทันที
