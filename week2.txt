#include <stdio.h> จำเป็นต้องมี # เสมอ และไม่ต้องมี ;
การเขียน main() คือถ้า เป็น int main() จะีเทิร์นค่า แต่ถ้าเป็น void main() จะไม่รีเทิร์นค่า

    การระบุตัวแปร ต้องมีการประกาศตัวแปรเสมอ คือ
int เก็บค่า จำนวนเต็ม
float เก็บค่า จำนวนจริง
char เก็บสัญลักษณ์หนึ่งตัวใน ASCII code
    
    ประเภทข้อมูล มีดังนี้
    char	1 byte
    unsigned char	1 byte	
    signed char	1 byte	
    int	4 byte
    unsigned int
    short	2 byte
    unsigned short
    long	4 byte
    unsigned long	4 byte	
    float	4 byte	
    double	8 byte	
    long double	10 byte	
    bool	1 bit	
**short เอาไว้ในการช่วยลดพื้นที่จัดเก็บของตัวแปรจำนวนเต็ม
**long ใช้ในการเพิ่มพื้นที่จัดเก็บ
**signed กำหนดตัวแปรเต็มให้เก็บค่าที่มีเครื่องหมายได้
**unsigned กำหนดจำนวนเต็มให้เก็บค่าเฉพาะค่าบวก

ค่าคงที่ จะมีคำสั่ง const นำหน้า และต้องกำหนดข้อมูลให้กัยค่าคงที่ด้วย เช่น
const float PI = 3.14; 
จำเป็นต้องมี semiclolon (;) เสมอ เพื่อแบ่งแยกคำสั่งโปรแกรม

การพิมพ์เลขฐาน 8 จะใช้ %o และจะมี 0 อยู่ข้างหน้า เช่น 0123
การพิมพ์เลขฐาน 16 จะใช้ %x จะมี 0x อยู่ข้างหน้าเช่น 0x123 
แต่หากเป็น %X จะเป็น 0X แทน เช่น 0X123

การที่จะแสดงว่าอักษรนี้เป็นอัขระตัวหนึ่งจะใช้ '' ในการแสดงเช่น 'A'
ส่วนค่าคงที่สตริงจะใช้ "" 
\n เป็นการใช้ในการขึ้นบรรทัดใหม่ 

ตัวดำเนินการทางคณิตศาสตร์ มีดังนี้
+ - * / %
/ จะไม่เอาเศษ
ส่วน % ไม่สามารถใช้กะ float และ double ได้ และจะเอาแค่ตัวเศษ

Bitwise
& และ
| หรือ
^ XOR
<< เลื่อนไปทางซ้าย
>> เลื่อนไปทางขวา
